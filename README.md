**svcc** is a cli tool to compile svelte files

## usage

to see help:

```sh
deno run --allow-net 'https://deno.land/x/svcc/cli.ts' --help
```

example:

```
deno run --allow-net 'https://deno.land/x/svcc/cli.ts' Hello.svelte
...
...
```
